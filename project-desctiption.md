# Florque

## Overview
Florque is an AI-powered project management autopilot designed to help individuals and small teams turn ideas into structured, executable work.

It transforms unstructured thoughts into clear project plans, organizes tasks into a connected work graph, and assists in executing work efficiently with AI.

---

## Core Idea
Traditional tools assume projects are already defined.

Florque solves the harder problem:

> Turning messy ideas into structured projects and helping execute them.

---

## Key Principles

### 1. Work as a Graph
Projects are not lists — they are networks of tasks, dependencies, and milestones.

Florque represents work as a **graph**, enabling:
- dependency tracking
- bottleneck detection
- impact analysis
- better prioritization

---

### 2. AI-Assisted Project Structuring
Users describe a project in natural language.

Florque generates:
- project structure
- tasks and subtasks
- logical execution flow

---

### 3. Automatic Task Breakdown
Large tasks are expanded into smaller, actionable steps using AI.

This reduces planning effort and improves execution clarity.

---

### 4. Execution Support
Florque continuously analyzes the project graph and suggests:
- next best tasks
- priority changes
- risk areas

---

### 5. Timeframe-Based Planning
Work is organized into **timeframes** (e.g. week, sprint) with defined capacity.

Florque helps:
- plan realistic workloads
- avoid overcommitment
- align tasks with available time

---

### 6. AI-Powered Project Operations
Operational steps (e.g. notifications, reports) can be automated.

Florque enables partial outsourcing of project operations to AI.
